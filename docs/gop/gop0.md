## Introduction

This GOP contains the index of all **Gnetcli Open Proposals**, known as **GOPs**.

GOPs are design and process documents for the **gnetcli** project. They are used to propose, discuss, and record significant changes to the project, including (but not limited to):

- public API and compatibility decisions;
- architectural changes;

**GOP numbers are assigned by the GOP editors.** Once assigned, GOP numbers are never changed. The version control history of GOP texts represents their historical record.

This index is updated as GOPs are created, accepted, rejected, or superseded.

## Topics

GOPs for specialist subjects are indexed by topic.

### Core GOPs

Core abstractions, primitives, expression system, command execution lifecycle.

### Drivers/Transports GOPs

Network OS/platform drivers, transport backends (SSH/Telnet/HTTP APIs), session handling, timeouts, retries.

### Documentation GOPs

Documentation structure and conventions, reference vs tutorials, examples policy.

### Release GOPs

Versioning policy, deprecation schedule, release cadence, support windows, changelog rules.
